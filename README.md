# AlgoLog

알고리즘 풀이를 기록, 공유, 질문 할수있는 웹페이지 입니다.

## 💻 프로젝트 소개

Next.js 와 MongoDB 를 이용한 토이 프로젝트입니다.

## ⏲️ 개발 기간

2024.02 - Present

### ⚙️ 개발 환경

<ul>
  <li>React.js@18.2.0, Next.js@v14.1.0, TypeScript, Redux Toolkit@9.1.0, Antd</li>
  <li>Env : Node.js@v18.17.0</li>
  <li>IDE : VScode</li>
  <li>DB : MongoDB </li>
  <li>CI/CD: Github Actions</li>
  <li>Deploy: Vercel</li>
</ul>
